Social Feed Aggregator
A Drupal 8 module that allows Facebook, Twitter and Instagram posts to be
pulled into a Social Post content type, on a scheduled basis.

Install
Use composer: composer require drupal/social_feed_aggregator

Requirements
If Composer isn't used to install the module, the following dependency
must be downloaded and placed in your Drupal project's vendor directory:
https://github.com/brynj-digital/social-feed
